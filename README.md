# PureTools — Free Browser-Based Tools

**No upload. No server. No signup. No watermark.**

PureTools is a growing collection of free online utilities covering finance, health, video, images, PDF, developer tools and more. Everything runs **entirely in your browser**. Files and inputs never leave your device — all processing happens locally using WebAssembly and modern Web APIs. Tools work offline after the first visit.

🌐 **[puretools.io](https://puretools.io)**

## Tech Stack

- **100% static** — HTML, CSS, vanilla JavaScript. Zero frameworks, zero backend.
- **WebAssembly** — FFmpeg.wasm for video tools, pdf-lib for PDF tools.
- **Service Worker** — offline support and instant repeat visits.
- **Cloudflare CDN** — global edge delivery.
- **No cookies, no analytics, no tracking.**

---

## Privacy Guarantee

Every tool on PureTools processes data **inside your browser**. There is no server-side processing:

- Video and image files are read via the File API and processed by WASM binaries running in a Web Worker.
- PDF operations use pdf-lib.js running in a Web Worker — no file bytes leave the tab.
- Password generation uses `crypto.getRandomValues()` — purely local.
- Hash computation uses the Web Crypto API — purely local.
- EXIF data is read using JavaScript — nothing is transmitted.
